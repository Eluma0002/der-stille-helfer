# Icons Folder

Add individual product icon images here.

## File Naming Convention:

Use lowercase, normalized German product names:

```
milch.png          → Milch, MILCH, milch
kaese.png          → Käse, Kaese, käse
apfel.png          → Apfel, Äpfel, äpfel
butter.png         → Butter
eier.png           → Eier, Ei
kartoffel.png      → Kartoffel, Kartoffeln
tomate.png         → Tomate, Tomaten
```

## Rules:
- Lowercase only
- Remove umlauts: ä→a, ö→o, ü→u, ß→ss
- Singular form preferred
- PNG or JPG format
- Recommended size: 256x256px or 512x512px

## Fallback:
If an icon is not found, the app automatically shows an emoji instead.
